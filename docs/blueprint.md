# **App Name**: MediChain

## Core Features:

- Role-Based Authentication: Secure login and signup for Admin, Doctor, and Patient roles using Firebase Authentication.
- Admin Dashboard: Manage doctors, patients, departments, and appointments.
- Doctor Dashboard: View assigned patients, appointments, and upload prescriptions as PDFs to Firebase Storage.
- Patient Dashboard: Book appointments, view prescriptions, lab reports, and medical history.
- Appointment Scheduling: Real-time appointment booking and updates.
- Electronic Medical Records: Securely store patient EMR data in Firestore.
- Symptom Checker Chatbot: AI-powered tool chatbot that checks possible disease of a person based on their symptoms and prior health conditions.

## Style Guidelines:

- Primary color: Calm blue (#4A8FE7) for a sense of trust and serenity.
- Background color: Light gray (#F0F4F8) for a clean and professional look.
- Accent color: Soft green (#73E0A9) for positive confirmations and highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif, for a modern but still slightly warm feel.
- Note: currently only Google Fonts are supported.
- Use FontAwesome for clear and consistent icons across the platform.
- Responsive design with a sidebar navigation for easy access to different sections.
- Subtle transitions and animations for a smooth user experience.
